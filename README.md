# shaped_image

A new Flutter package for showing images in different shapes

## Getting Started
To use this package, add `shaped_image` as a dependency in your `pubspec.yaml` file

```yaml
dependencies:
  flutter:
    sdk: flutter
  shaped_image: ^0.0.3
```
## Example 

 ```dart
            ShapedImage(
               imageTye: ImageType.ASSET,
               path: 'assets/test.jpeg',
               shape: Shape.SHAPE07,
               width:200,
               height:200
              ),      
 ```
